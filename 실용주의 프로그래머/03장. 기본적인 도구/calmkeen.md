## 20. 코드 생성기



> 코드를 작성하는 코드를 작성하라



코드 생성기의 두가지 유형

1. 동적 코드는 결과를 내기 위해 한 번만 실행된다.
2. 능동적 코드 생성기는 코드 생성이 필요할 떄마다 작동한다.





#### 수동적 코드 생성기



- 새 소스 파일 생성
- 프로그래밍 언어간 일회용 변환을 수행하기
- 런타임에 계산하기엔 비용이 많이 드는 참조 테이블과 여타 자원을 생성하기



#### 능동적 코드 생성기

어떤 지식을 단 하나의 형태로만 만들어놓고 애플리케이션이 필요로 하는 온갖 형식을 변환할 수 있다.

정보를 중첩시키지 말고 코드 생성기를 사용하라. 소스 파일에서 이런 정보들을 파싱해 가져와서 다른 언어의 코드를 생성하는 데 사용할 수도 있다.



하지만 코드 생성기가 꼭 필요하지만은 않다.
