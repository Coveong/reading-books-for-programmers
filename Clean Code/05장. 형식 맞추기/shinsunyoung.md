# 5장 형식 맞추기

1. 적절한 행 길이 유지하기
    - 파일도 작은 파일이 더 이해하기 쉽다.
    - 개념은 빈 행으로 분리하는 게 좋다. (패키지 선언부, import문)
    - 서로 밀접한 개념은 세로로 가까이 두는게 좋다.
        - **변수**는 사용하는 위치에 최대한 가까이 선언하는 게 좋다.
        - **인스턴스** 변수는 클래스 맨 처음에 선언하는 게 좋다.
        - **종속 함수**는 세로로 가까이 배치하는 게 좋다.
        - **개념적 유사성**이 높을수록 코드를 가까이 배치하는 게 좋다.
2. 가로 형식 맞추기
    - 짧을수록 좋다.
    - 가로 정렬은 안 하는 게 좋다.
    - 들여쓰기는 있는 게 좋다.
