# 개발 서적 읽기

## 규칙 📝

1. 공유미팅에서 한 주 동안 읽을 책의 분량을 정합니다. (장 단위)
1. 정해진 분량의 책을 열심히 읽고 틈틈이 정리합니다.
1. 다음 공유미팅 시작 전까지 정리한 내용을 이 레포지토리에 업로드합니다.

🚨 위 규칙을 지키지 못할 시 벌금(5,000원/장)이 부과됩니다!!!

## 일정

공유미팅은 매주 주말 중 하루동안 진행합니다.
